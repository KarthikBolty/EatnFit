# EatnFit

A Full Stack Fitness & Nutrition Tracking Web Application developed using **Python, Django, Django REST Framework, React.js, HTML, CSS, Bootstrap, and MySQL**.

## Features

- User Registration & Login
- Calorie Tracking
- BMI Calculator
- Body Weight Monitoring
- Personalized Diet Planner
- Dashboard with Daily Statistics
- Responsive User Interface
- RESTful API Integration
- Database Management with Django ORM

## Tech Stack

### Frontend

## Installation

### Clone the Repository

```bash
git clone https://github.com/KarthikBolty/EatnFit.git
```

### Backend Setup

```bash
cd EatnFit

python -m venv env

env\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
